CSV Load, Sort, Report

Scripts to Load, Sort, and Report on a CSV file

Installation Instructions

These scripts simply require Python to run. They mainly make use of the Python standard library. To use them, simply clone this repository, or download the code above.

  • The vanilla_approach.py script has no external dependancies.
  • The database_approach.py script does not require any external libraries to run, but does require Sqlite3 to be installed in order to use its output.
  • The pandas_approach.ipynb notebook depends on IPython Notebook, and the Pandas data analysis library.

Overview

This is a very open ended problem, and each approach represents a slightly different mentality towards coming up with a solution, and involves various benefits and tradeoffs, as discussed below.

Vanilla Approach

This is the most straightforward solution to the problem. It is object-oriented, and strives to keep things simple, and solve the problem as specified. The data structures chosen focus specifically on the 2 operations required by the spec, at the expense of flexibility.

Running this script can be done as follows:

~>  python vanilla_approach.py --data INPUT_CSV_FILE

To see the available command line options, use the -h or --help flag.

Database Approach

This approach is a thin wrapper over Sqlite3. The rationale here is simple. We want to perform queries, and extract relations from the input data. Instead of maintaining custom software to manage the data, and using Python as a data-munging DSL, why not leverage an existing solution (SQL), and enjoy the efficiency and reliability of a mature technology.

This script simply performs the loading of the input data into a sqlite database. It also has helper methods for the 2 operations required by the spec, but the idea is that the underlying sqlite db file can be used directly as well.

Running this script can be done as follows:

~>  python database_approach.py --db DB_OUTPUT_FILE --data INPUT_CSV_FILE

To see the available command line options, use the -h or --help flag.

Pandas Approach

Using IPython Notebook allows for easy interaction with, and analysis of the input data. The advantage of this approach is that it is interactive, and very conducive to exploration. Additionally, Pandas has a lot of convenience functionality built in, which makes playing with the data almost trivial.

A read-only copy of the notebook can be found here. To run the notebook locally, after installing the relevant libraries, run:

~> ipython notebook pandas_approach.ipynb
